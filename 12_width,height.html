width 특징

  • width는 부모요소의 크기를 그대로 사용한다(상속됨)
  • width값을 작성하는 경우 입력한 크기대로만 적용된다(고정값)
  • 인라인 요소의 width는 컨텐츠 가로크기만큼만 width가 적용되며 사용자가 입력하는 width는 적용되지 않는다
  • width에 상대단위 (%,em)을 입력하는 경우 부모요소의 크기를 기준으로 값이 적용
