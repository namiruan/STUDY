ID와 CLASS

id 선택자

요소에 id속성을 생성하고, id이름을 부여

선택자는 id이름을 사용하고 요소명#선택자 이름과 같이 사용(요소명 생략가능)

id선택자는 문서에서 오직 한번만 사용가능(id이름을 서로다른 요소에서 중복사용불가)

사용예제

아이디 선택자 테스트

아이디 선택자 중복사용 불가

class 선택자

요소에 class 속성을 생성하고 class 이름을 부여한다

선택자는 class 이름을 사용하고 요소명.class 이름과 같이 사용한다

class선택자는 서로 다른 요소에서 중복사용 가능하다

class선택자는 하나의 class속성에 2개 이상의 class이름을 부여할 수 있다
