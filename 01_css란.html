CSS

CSS정의

CSS는 HTML문서를 디자인기기위한 언어

CSS는 독립적인 언어가 아니라 HTML문서안에서만 동작

CSS는 레벨1, 레벨2, 레벨3까지 나왔으며 레벨3은 현재 진행형(표준안 발표전)임

CSS는 브라우저별로 지원하는 스타일, 또는 각 브라우저 버전에 따라 차이가 발생 (해결= 크로스 브라우징)

CSS사용하기

HTML문서에서 CSS를 적용할 때 다음과 같은 세가지 방법을 사용한다

CSS는 HTML문서를 제어할 때 element(요소)단위로만 제어한다

  • 인라인 스타일(inline style) : 스크립트에 적용되는 스타일은 인라인 스타일로 적용
  • 내부 스타일(internal style) : 가장 많이 사용하나 실무에 적합하지 않음
  • 외부 스타일(external style) : 실무에 적합하고 유지보수에 용이함
