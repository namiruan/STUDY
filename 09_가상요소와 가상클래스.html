가상요소와 가상클래스

가상요소와 가상클래스는 요소를 선택할 때 특정이벤트(상황)나 특정영역을 선택하기 위해 사용하는 선택자

선택자로 사용시 요소명:가상요소(클래스)이름과 같이 사용

가상요소

  • first-letter : 문단의 첫글자
  • first-line : 문단의 첫줄
  • first-child : 첫번째 자식요소(중요)
  • before : 요소안의 컨텐츠 앞(중요)
  • after : 요소안의 컨텐츠 뒤 (중요)

    비포에프터 사용하기

    가상클래스(중요)

    • link : 하이퍼링크가 적용된 요소
    • visited : 방문한 요소
    • hover : 요소에 마우스가 올라갔을 때
    • active : 요소가 활성화 되었을 때 (하이퍼링크)
    • focus : 요소가 포커스 되었을 때
